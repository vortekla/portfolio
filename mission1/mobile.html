Développeur mobile

Les développeurs mobiles se concentrent sur le développement d'applications pour appareils mobiles . Ils sont également connus sous le nom de développeurs Android ou iOS s'ils travaillent sur une seule plateforme, mais il est courant que les développeurs travaillent sur les deux. Les tâches typiques incluent : discuter des exigences des clients et de ce qu'ils aimeraient que l'application fasse.
